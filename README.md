# BitBridge

Seamless, secure file transfer across Desktop, Android, iOS, and the browser.

## Overview

BitBridge makes moving files between your devices—and the people around you—fast and effortless. Send directly to nearby devices or generate a public share link with an optional password for anyone, anywhere.

## Key features

- Cross‑platform: Desktop (Windows, macOS, Linux), Android, iOS, and Web
- Nearby transfer: discover and send to devices around you on the same network
- Public link sharing: create a sharable link for any file or folder
- Optional password protection for shared links
- No cables, no accounts required
- Designed for reliability with large files and spotty networks

## How it works

1) Choose files to send.
2) Pick recipients nearby, or create a public link to share broadly.
3) Optionally set a password for the link. Share the link (and password, if set) with your recipient.

## Typical uses

- Share with teammates in a meeting without email or chat uploads
- Move photos and videos between your phone and laptop
- Send files to a group using a single link with optional password

## Security and privacy

- Nearby transfers stay device‑to‑device on your local network
- Public links can be protected with a password
- You control when and with whom you share

## Get started

- Install the app on your devices or open the web client
- Open BitBridge on all participating devices
- Send nearby or generate a link—your choice
